## üé® Customization

To personalize the portfolio with your information:

1. **Personal Details** - Update name, tagline, and bio in `index.html`
2. **Skills** - Modify the skills sections with your technologies
3. **Projects** - Add your projects with descriptions and GitHub links
4. **Contact Information** - Update email, GitHub, and LinkedIn URLs
5. **Styling** - Adjust colors and design in `style.css` using CSS variables
